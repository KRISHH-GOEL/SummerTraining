# Lesson 02 – Linear Algebra & Statistical Functions

## 📌 Objective

The objective of this lesson was to explore NumPy's powerful linear algebra and statistical capabilities. These tools form the mathematical foundation of Data Science, Machine Learning, and Artificial Intelligence.

This lesson focused on performing matrix operations, solving linear algebra problems, and extracting meaningful statistical insights from numerical datasets.

---

## 📚 Topics Covered

### Linear Algebra

- Dot Product (`np.dot()`)
- Matrix Multiplication (`np.matmul()` / `@`)
- Matrix Determinant (`np.linalg.det()`)
- Eigenvalues and Eigenvectors (`np.linalg.eig()`)

### Statistical Functions

- Mean (`np.mean()`)
- Median (`np.median()`)
- Standard Deviation (`np.std()`)
- Variance (`np.var()`)
- Percentiles (`np.percentile()`)
- Correlation Coefficient (`np.corrcoef()`)

---

## 🎯 Learning Outcomes

After completing this lesson, I can:

- Compute the dot product of vectors.
- Perform matrix multiplication.
- Calculate the determinant of a matrix.
- Find eigenvalues and eigenvectors.
- Analyze numerical datasets using statistical functions.
- Measure spread using variance and standard deviation.
- Calculate percentiles.
- Measure relationships between datasets using correlation coefficients.
