# Lesson 02 – Functions, Comprehensions & File Handling

## 📌 Objective

The goal of this lesson was to understand how to write reusable code using functions, simplify data processing with comprehensions, and work with files for reading and writing data.

---

## 📚 Topics Covered

### Functions

* Defining functions using `def`
* Function parameters and arguments
* Returning values using `return`

### Variable-Length Arguments

* `*args`
* `**kwargs`

### Lambda Functions

* Anonymous functions
* Using lambda expressions with built-in functions

### Comprehensions

* List Comprehensions
* Dictionary Comprehensions
* Generator Expressions

### File Handling

* Opening files
* Reading text files
* Writing text files
* Appending to files
* Using the `with` statement

### CSV Files

* Reading CSV files
* Writing CSV files

---

## 🎯 Learning Outcomes

After completing this lesson, I can:

* Create reusable functions.
* Pass values into functions and return results.
* Work with an unknown number of arguments using `*args` and `**kwargs`.
* Write concise code using lambda functions.
* Generate lists, dictionaries, and generators efficiently using comprehensions.
* Read from and write to text files.
* Safely manage files using the `with` statement.
* Read and write CSV files using Python's built-in `csv` module.
