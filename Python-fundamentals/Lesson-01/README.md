# Lesson 01 – Python Fundamentals

## 📌 Objective

The goal of this lesson was to understand the basic building blocks of Python programming.

---

## 📚 Topics Covered

* Variables
* Data Types
* Type Casting
* Input and Output
* Operators
* Conditional Statements (`if`, `elif`, `else`)
* Loops (`for` and `while`)
* `break`
* `continue`
* `pass`

---

## 🎯 Learning Outcomes

After completing this lesson, I can:

* Declare and use variables.
* Work with different Python data types.
* Perform arithmetic, comparison, logical, assignment, and membership operations.
* Accept user input and display output.
* Write decision-making programs using conditional statements.
* Implement repetition using `for` and `while` loops.
* Control loop execution using `break`, `continue`, and `pass`.
