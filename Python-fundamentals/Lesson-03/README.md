# Lesson 03 – Object-Oriented Programming (OOP) & Exception Handling

## 📌 Objective

The objective of this lesson was to understand Object-Oriented Programming (OOP) in Python, learn how to design reusable and maintainable programs using classes and objects, explore Python's special (dunder) methods and decorators, and handle runtime errors using exception handling.

---

## 📚 Topics Covered

### Object-Oriented Programming (OOP)

- Classes
- Objects
- Constructors (`__init__`)
- Instance Variables
- Instance Methods

### OOP Principles

- Inheritance
- Polymorphism
- Encapsulation

### Magic (Dunder) Methods

- `__repr__`
- `__len__`
- `__iter__`

### Decorators

- `@staticmethod`
- `@classmethod`
- `@property`

### Exception Handling

- `try`
- `except`
- `else`
- `finally`

### Custom Exceptions

- Creating user-defined exceptions
- Raising exceptions using `raise`

---

## 🎯 Learning Outcomes

After completing this lesson, I can:

- Create classes and objects.
- Initialize objects using constructors.
- Implement inheritance to reuse code.
- Demonstrate polymorphism through method overriding.
- Apply encapsulation to protect object data.
- Use Python's built-in magic methods.
- Create static methods, class methods, and properties.
- Handle runtime errors gracefully using exception handling.
- Define and raise custom exceptions.
