# Lesson 01 – Pandas Basics, Data Loading & Data Cleaning

## 📌 Objective

The objective of this lesson was to understand the fundamentals of the Pandas library, which is widely used for data manipulation and analysis in Python.

This lesson focused on creating Series and DataFrames, loading datasets from different file formats, selecting data using indexing techniques, and performing essential data cleaning operations.

---

## 📚 Topics Covered

### Introduction to Pandas

- What is Pandas?
- Why Pandas is used in Data Science
- Importing Pandas

### Series

- Creating a Series
- Series Attributes
- Data Types (`dtype`)

### DataFrame

- Creating DataFrames
- DataFrame Attributes
- Data Types (`dtypes`)

### Loading Data

- `pd.read_csv()`
- `pd.read_excel()`
- `pd.read_json()`
- `pd.read_sql()`

### Data Selection

- `loc[]`
- `iloc[]`
- Boolean Masking

### Data Cleaning

- `dropna()`
- `fillna()`
- `drop_duplicates()`
- `astype()`
- `rename()`

---

## 🎯 Learning Outcomes

After completing this lesson, I can:

- Create and manipulate Series and DataFrames.
- Inspect datasets using DataFrame attributes.
- Load data from CSV, Excel, JSON, and SQL sources.
- Select rows and columns using `loc` and `iloc`.
- Filter data using Boolean masking.
- Handle missing values.
- Remove duplicate records.
- Change data types.
- Rename columns efficiently.
